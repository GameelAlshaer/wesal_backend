    Take action on actual reports

    requires authentication

    This function to take action on report with input 'report_id' (int) required to do report on it and input 'action_type' (int) required to know what action to do action_type can be = 1 || 2 || 3 || 4 ( (1) for no action or (2) for removing the report from the user or (3) for banning user or (4) for temp banning user for limited time (weekly checks to remove the ban from him))
